# 🚀 MWIT DS Tool (LearnLab Bridge Public) - v0.4.5

ยินดีต้อนรับสู่โปรเจกต์ **MWIT DS Tool** ซึ่งเป็นเครื่องมือบริหารจัดการสภาพแวดล้อมการเรียนรู้ JupyterLab สำหรับโรงเรียนมหิดลวิทยานุสรณ์

## 📦 ความสามารถหลัก
- **Custom JupyterHub Authenticator**: ระบบจัดการสิทธิ์การเข้าถึง Lab
- **Infrastructure as Code**: ตัวอย่างการตั้งค่า Docker สำหรับการทำ Learning Platform
- **Localhost Ready**: รองรับการใช้งานในวง LAN และการทำ Lab ออฟไลน์

## 🛠 การเริ่มต้นใช้งาน
1. คัดลอกโปรเจกต์: `git clone https://git.mwit.ac.th/service/mwit-ds-tool.git`
2. ตั้งค่าไฟล์คอนฟิก: คัดลอก `.env.example` เป็น `.env` และปรับค่าตามต้องการ
3. เริ่มต้นระบบ: รันสคริปต์ `./scripts/up.sh` (หรือ `up.ps1` สำหรับ Windows)
